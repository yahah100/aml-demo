Visualization and Analysis of Machine Learning Results

Project Overview

This project showcases various visualizations and analyses from the Applied Machine Learning lecture. The visualizations include different plots that help understand the data and model results better.

Visualizations

This visualization shows analysis of a welding sample, providing insights into the welding quality parameters.

Exercise 2: Interactive Welding Sample Visualization

An interactive plotly visualization that allows exploring the welding sample data in detail.

Exercise 3: 3D Cluster Analysis

This 3D visualization shows the clustering results with 3 distinct clusters.

Exercise 3: 3D Quality Analysis

This 3D visualization demonstrates the quality analysis in three-dimensional space.
